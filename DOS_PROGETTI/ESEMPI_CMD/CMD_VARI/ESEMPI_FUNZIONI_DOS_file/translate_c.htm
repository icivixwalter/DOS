La funzione restorePersistentVars legge le variabili di nuovo da un file.

4 4         Time Of Day and runtime measurements - :getTod, :getHHMMSSDD, :Tod2Str, :TodDiffNow Temporale delle misure di Giorno e di esecuzione -: getTod,: getHHMMSSDD,: Tod2Str,: TodDiffNow

The getTod function returns a Time Of Day (Tod) integer value with a precision of 1/100 th of a second. La funzione restituisce un getTod Ora del giorno (Tod) valore intero con una precisione di 1/100 di secondo.   Together with the other functions this can be used for time capture calculations like runtime measurements. Insieme con le altre funzioni questo può essere utilizzato per i calcoli catturare il tempo, come le misure di esecuzione.

The getHHMMSSDD function allows extracting the hours, minutes, seconds, and 1/100 th of seconds from a Tod integer. La funzione getHHMMSSDD permette di estrarre le ore, minuti, secondi e 1/100 di secondo da un numero intero Tod.

The Tod2Str function allows to freely format a Tod integer into a string, ie put the seconds, hours, minutes where you want them. La funzione Tod2Str permette di formattare liberamente un numero intero Tod in una stringa, cioè mettere i secondi, ore, minuti dove vuoi.

8 8         Float <-> Integer Conversion - :F2I, :I2F Float <-> Integer conversione -: F2I,: i2F

Una stringa memorizzare un valore in virgola mobile può essere convertito in un numero intero utilizzando la funzione F2I.   To preserve precision the float will be multiplied by 10, 100, 1000, or more so that the fractions move into the integer part of the value.
